# UI Direction (Nature Motif)

## 1. デザインテーマ
"Quiet Growth Garden"

静かな背景の中で、学習の積み上げが有機的に育っていく表現を採用する。
AI生成っぽい均一デザインを避け、タイポ・余白・色の抑揚で個性を作る。

## 2. ビジュアル原則
- 無機質なカード並べを避ける
- 余白を広く取り、視線の流れを設計する
- 色は土・葉・水・朝焼けを参照したパレットに限定
- 主要コンポーネントに丸みを使いすぎない（甘くなりすぎ防止）

## 3. カラーパレット（初期案）
- Soil: #3D2C24
- Moss: #5A7D4D
- Leaf: #7FA36B
- Mist: #EDE9E3
- Water: #7BA7B6
- Sunrise: #D98E5F

## 4. タイポグラフィ（初期案）
- Heading: "Fraunces"（セリフ、存在感）
- Body/UI: "Manrope"（可読性）

※ 実装時に Web フォント読み込みを最小限にし、表示速度と両立する。

## 5. モーション方針
- 初回表示でヒートマップを行単位で軽くフェードイン
- 記録追加時、該当日のセルに短い拡散アニメーション
- 過度なアニメーションは避け、達成時のみ強調

## 6. レイアウト方針
- Desktop: 左に週次サマリ、右に記録入力 + ヒートマップ
- Mobile: 上から「今日の記録」「ヒートマップ」「週次サマリ」
- 最重要アクション（記録追加）を常に 1 タップ以内に配置
