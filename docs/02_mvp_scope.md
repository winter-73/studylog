# MVP Scope (v0.1)

## 1. MVPでやること
- 勉強記録を追加できる
- 月表示の継続ヒートマップを表示できる
- 各記録に成長軸タグをつけられる
- 週次サマリ（今週の合計時間、主な成長軸）を表示する

## 2. MVPでやらないこと
- SNS機能
- 他ユーザー共有
- 高度な推薦
- 複雑なゲーミフィケーション

## 3. ユースケース（最小）
1. ユーザーが勉強後に記録を追加する
2. カレンダーで継続状況を確認する
3. 週次で成長軸ごとの偏りを振り返る

## 4. 成長軸（初期定義）
- Understanding: 理解
- Building: 実装
- Output: 発信

## 5. 成功指標（MVP検証）
- 7日継続率
- 1日1回以上の記録率
- 週次サマリ閲覧率
