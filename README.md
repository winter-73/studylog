# StudyLog

勉強記録を「継続」と「成長」で可視化する Web アプリです。

## コンセプト
- GitHub の草のように、日々の積み上げをひと目で見える化
- 単なる記録ではなく、やった行動が「何の成長につながったか」を紐づけ
- 自然モチーフ（庭・季節・水循環）で、継続するほど景色が育つ体験を目指す

## 開発方針
- いきなり全部は作らない
- 設計 -> 最小実装 -> 振り返り -> 拡張を反復
- ポートフォリオとして、判断理由と設計の筋道を残す

## まず作るもの
MVP v0.1 として、以下を実装する。
- 勉強記録の追加（学習時間、カテゴリ、メモ）
- 日次の継続ヒートマップ表示
- 成長軸（例: 理解、実装、アウトプット）への紐づけ表示
